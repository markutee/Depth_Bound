# 2D Mining Game (Godot)

This game is made for study purposes.

##
This project is a 2D mining game developed using the Godot game engine as part of a thesis project. The goal of the project was to design and implement a playable prototype while exploring the game development process, including agile methods and iterative prototyping.

## Features

- Player movement in a 2D mining environment
- Resource collection and mining mechanics
- Inventory system
- Shop system for upgrading tools
- Multiple map layers with increasing depth
- Darkness mechanic that limits visibility

## Technologies Used

- Godot Engine
- GDScript
- Git (version control)

## How to Run

1. Download or clone this repository
2. Open the project in Godot Engine
3. Run the main scene

## Project Context

This game was developed as part of a thesis focusing on:
- Indie game development
- Agile development methods
- Iterative prototyping

The result is a playable prototype demonstrating core game mechanics and development practices.
